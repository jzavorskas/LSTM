# LSTM
Companion code for biorxiv preprint and ACS Omega manuscript in review "ao-2025-11599".

All viral propagation and brlA simulation code can be found in their respective folders. Raw brlA rt-qPCR data is available, as well as the supplemental tables available with the manuscript.

A persistent code repository is available at: https://doi.org/10.5281/zenodo.18222496
